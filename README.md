## ü§ù Contributing

### For Analysts & Researchers
1. **Add New Analysis Reports**: Use our [report template](data/templates/project_report_template.md) to create insights
2. **Enhance Existing Reports**: Improve depth, add new insights, or update findings
3. **Create Visual Diagrams**: Develop Excalidraw/Tldraw compatible SVG diagrams

### For Developers
- **Technical Contributions**: See [Scripts Documentation](scripts/README.md) for development guidelines

### Analysis Contribution Process
1. Choose an analysis focus from our solutions framework
2. Create folder: `docs/your-analysis-name/`
3. Generate: `analysis.md` (report) + `diagrams.svg` (visuals)
4. Add entry to the appropriate table in this README
5. Submit pull request with clear description

### Quality Guidelines
- ‚úÖ Use evidence-based insights from the 1,032 project dataset
- ‚úÖ Include actionable recommendations for entrepreneurs/policymakers
- ‚úÖ Provide both analysis.md and diagrams.svg files
